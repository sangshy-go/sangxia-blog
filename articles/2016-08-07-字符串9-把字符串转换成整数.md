### 题目描述

将一个字符串转换成一个整数，要求不能使用字符串转换整数的库函数。

思路：

1）正负号；

2）非法字符输入；

3）整数溢出。

```
public class Solution {
    public int StrToInt(String str) {
        if(str == null || str.length() <= 0){
            return 0;
        }
        
        
        char[] strArr = str.toCharArray();
        int sum = 0;
        int index = 0;
        boolean minus = false;
        if(strArr[0] == '+' || strArr[0] == '-'){
            index++;
            if(strArr[0] == '-'){
                minus = true;
            }
        }
        int count = 0;
        int flag = minus? -1 : 1;
        for(int i = strArr.length - 1; i >= index; i--){
            char c = strArr[i];
           if(c >= '0' && c <= '9'){
               //从前往后计算
               //sum = sum * 10 + flag * (c - '0');
               //从后往前计算
                sum += flag * Math.pow(10, count++) *  (c - '0');
               //整数溢出Integer.MAX_VALUE:0x7FFFFFFF
            	if(!minus && sum > 0x7FFFFFFF){
                	return 0x7FFFFFFF;
            	}
               //整数溢出Integer.MIIN_VALUE:0x80000000
           		if(minus && sum < 0x80000000){
                	return 0x80000000;
           		}
           }else{
               //非法字符
               return 0;
           }
           
        }
        return sum;
    }
}
```
