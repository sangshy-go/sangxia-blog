第一次使用GitHub,并上传自己的一个项目。下面简单介绍自己的使用过程。

### 

### 一 生成自己的公钥与私钥

1
打开Git Bash(git客户端，下载安装即可)，输入ssh-keygen -t rsa -C "自己的邮箱地址@XXX.com" 

自己的公钥与私钥；

2
生成一个指定的文件，里面会生成公钥、私钥到以下文件夹下。

id\_rsa是私钥，id\_rsa.pub是公钥，打开公钥id\_rsa.pub内内容并复制，等下要用到；

![](https://img-blog.csdn.net/20161227175058626?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

3浏览器进入自己的github，打开设置，进入ssh
and GPG keys

![](https://img-blog.csdn.net/20161227175006407?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

4点击NEW
ssh key，自己填个标题，下面内容复制前面打开的公钥，最后添加

![](https://img-blog.csdn.net/20161227175130236?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

5 可以用ssh
-T git@github.com测试自己是否可以连接成功了，中间有个输入需要输入yes，后面可以看到Hi XXXX..... access 表示成功。

### 三，上传本地文件到网页GitHub服务端

 1 在本地建一个文件夹，内是要上传的项目，如我要上传的项目是MybatisTestDemo1的内容。

![](https://img-blog.csdn.net/20161227175416627?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

2 点进去，空白处点击右键，选择Git bash here,打开客户端。

![](https://img-blog.csdn.net/20161227175527565?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

3 分别执行指令  ：git
init

   
                                  git add .

git
commit -m "提交文件" （双引号内容无所谓）

![](https://img-blog.csdn.net/20161227175648160?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

上面三个指令执行完后，代表本地仓库建立好了。

4 关联github仓库，即要知道将本地代码上传到哪个地方去：

执行指令git
remote add origin “git@github.com:wangjiax9/beautifulDay.git”(括号内内容即为之前需要注意的SSH后的项目地址)。

这里的双引号内的地址跟我下面的黑图地址不一样，因为这个是复制别人的，图中地址才是我的项目的地址。

![](https://img-blog.csdn.net/20161227180310623?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)
