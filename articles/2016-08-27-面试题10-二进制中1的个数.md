### 题目描述

输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

一、比较容易懂的方法：该数与1 位与运算，如果该数的某位数为1，与1运算后的结果不为0（如10001 & 0001 ==0001, 0001 !=0）

     如果该数的某位数为0，与1运算后的结果为0（如10000 & 0001 ==0000, 0000 !=0）

```
public class Solution {
    public int NumberOf1(int num) {
        if(num==0)
            return 0;
        int count=0;
        int flag=1;
        int result=0;
        while(flag!=0){
            result=num & flag;
            if(result!=0){
                count++;
            }
        	flag=flag<<1;
        }
   		 return count;
	}
}
```
