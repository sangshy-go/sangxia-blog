### 题目描述

输入一个int型的正整数，计算出该int型数据在内存中存储时1的个数。

  

###### 输入描述:

```
 输入一个整数（int类型）
```

  
  

###### 输出描述:

```
 这个数转换成2进制后，输出1的个数
```

  

###### 输入例子:

```
5
```

  

###### 输出例子:

2

```
import java.util.Scanner;

public class Main{
    private static int countNumsOfOne(int num){
        int count=0;
        while(num!=0){
            int result=num % 2;
            if(result==1)
                count++;
            num=num/2;
        }
        return count;
    }
    
    public static void main(String[] args){
        Scanner sc=new Scanner(System.in);
        while(sc.hasNext()){
            int num=Integer.parseInt(sc.nextLine());
            System.out.print(countNumsOfOne(num));
        }
        sc.close();
    }
}
```
