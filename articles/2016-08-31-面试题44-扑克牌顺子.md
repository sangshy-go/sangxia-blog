### 题目描述

LL今天心情特别好,因为他去买了一副扑克牌,发现里面居然有2个大王,2个小王(一副牌原本是54张^\_^)...他随机从中抽出了5张牌,想测测自己的手气,看看能不能抽到顺子,如果抽到的话,他决定去买体育彩票,嘿嘿！！“红心A,黑桃3,小王,大王,方片5”,“Oh My God!”不是顺子.....LL不高兴了,他想了想,决定大\小 王可以看成任何数字,并且A看作1,J为11,Q为12,K为13。上面的5张牌就可以变成“1,2,3,4,5”(大小王分别看作2和4),“So Lucky!”。LL决定去买体育彩票啦。
现在,要求你使用这幅牌模拟上面的过程,然后告诉我们LL的运气如何。为了方便起见,你可以认为大小王是0。

```
import java.util.Arrays;

public class Solution {
    public boolean isContinuous(int [] nums) {
		if(nums==null || nums.length!=5)
            return false;
        //1 排序,如果有大小王，令其为0，排在第一位
        Arrays.sort(nums);
        //2 统计大小王个数
        int numsOfJokers=0;
        for(int i=0;i<nums.length;i++){
            if(nums[i]==0)
                ++numsOfJokers;
        }
        
        //3 统计摸到的牌的间隔数
        int gaps=0;
        //肯定是从非0的数开始比较与后一个数间隔；
        //例如有2个0，则在数组中应该从第二个数开始判断
        int small=numsOfJokers;
        int big=small+1;
        while(big<nums.length){
            //前后两个数相等自然不能成为顺子
            if(nums[small]==nums[big])
                return false;
            //如：6与4间隔1个数5
            gaps+=nums[big]-nums[small]-1;
           	small++;
            big++;
        }
        //间隔数小于等于王的个数即为顺子
        return (gaps<=numsOfJokers)?true:false;
        
    }
}
```
