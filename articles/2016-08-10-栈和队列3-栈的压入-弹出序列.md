题目描述

输入两个整数序列，第一个序列表示栈的压入顺序，请判断第二个序列是否为该栈的弹出顺序。假设压入栈的所有数字均不相等。例如序列1,2,3,4,5是某栈的压入顺序，序列4，5,3,2,1是该压栈序列对应的一个弹出序列，但4,3,5,1,2就不可能是该压栈序列的弹出序列。

```
    public boolean isPopOrder(int[] pushA,int[] popA){
        
        
        boolean flag=false;
        ArrayList<Integer> assistList=new ArrayList<Integer>();
    
        
        if(pushA.length>0 && popA.length>0 && pushA.length==popA.length){
            
            //当前 popA中的数
            int pop=0;
            //下一个弹出的元素
            int nextPop=0;
            //pushA当前元素
            int push=0;
            //下一个压入的元素
            int nextPush=0;
            int len=pushA.length;
            int index=-1;
            while(nextPop-pop<len){//要求的顺序
                //下一个弹出的数字不在栈顶，把压栈序列中还没有入栈的数字压入辅助栈
                while(assistList.size()==0 || assistList.get(index)!=popA[nextPop]){
                    if(nextPush-push==len) break;
                    assistList.add(pushA[nextPush]);
                    index++;
                    nextPush++;
                }
                //确认
                if(assistList.get(index)!=popA[nextPop]) break;
                //当该元素及之前元素加入完了之后，就弹出去，指向popA中下个顺序的元素
                assistList.remove(index--);
                nextPop++;
            }
            if(assistList.size()==0 && nextPop-pop==len){
                flag=true;
            }
            
        }
        return flag;
        
    }
```

  

----

【思路】借用一个辅助的栈，遍历压栈顺序，先讲第一个放入栈中，这里是1，然后判断栈顶元素是不是出栈顺序的第一个元素，这里是4，很显然1≠4，所以我们继续压栈，直到相等以后开始出栈，出栈一个元素，则将出栈顺序向后移动一位，直到不相等，这样循环等压栈顺序遍历完成，如果辅助栈还不为空，说明弹出序列不是该栈的弹出顺序。

举例：

入栈1,2,3,4,5

出栈4,5,3,2,1

首先1入辅助栈，此时栈顶1≠4，继续入栈2

此时栈顶2≠4，继续入栈3

此时栈顶3≠4，继续入栈4

此时栈顶4＝4，出栈4，弹出序列向后一位，此时为5，,辅助栈里面是1,2,3

此时栈顶3≠5，继续入栈5

此时栈顶5=5，出栈5,弹出序列向后一位，此时为3，,辅助栈里面是1,2,3

….

依次执行，最后辅助栈为空。如果不为空说明弹出序列不是该栈的弹出顺序。

  

```
import java.util.*;

public class Solution {
    public boolean IsPopOrder(int [] pushA,int [] popA) {
       if(pushA == null || popA == null)
           return false;
        
        Stack<Integer> stack = new Stack<>();
        int popIndex = 0;
        for(int i = 0; i < pushA.length; i++){
            stack.push(pushA[i]);
            while(!stack.isEmpty() && stack.peek() == popA[popIndex]){
                popIndex++;
                stack.pop();
            }
        }
        
        return stack.isEmpty();
    }
}
```
