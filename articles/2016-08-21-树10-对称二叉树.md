### 题目描述

请实现一个函数，用来判断一颗二叉树是不是对称的。注意，如果一个二叉树同此二叉树的镜像是同样的，定义其为对称的。

```
/*
public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;

    public TreeNode(int val) {
        this.val = val;

    }

}
*/
/**
 *自定义前序遍历来遍历一颗二叉树后再与其前序遍历比较，相同则是镜像的。
 	如何自定义：先右后左
 */
public class Solution {
    boolean isSymmetrical(TreeNode pRoot)
    {
        if(pRoot==null)
            return true;
        return isSymmetricalCore(pRoot,pRoot);
    }
    
    private boolean isSymmetricalCore(TreeNode pRoot,TreeNode pRootA){
        if(pRoot==null && pRootA==null)
            return true;
         if(pRoot==null || pRootA==null)
            return false;
        if(pRoot.val!=pRootA.val)
            return false;
        return isSymmetricalCore(pRoot.left,pRootA.right) 
                &&isSymmetricalCore(pRoot.right,pRootA.left);
        
    }
}
```
