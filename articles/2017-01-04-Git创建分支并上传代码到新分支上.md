### Git创建分支并上传代码到新分支上

#### 一  创建分支：

我是在Github上项目下直接创建的新分支，此处以后再补充；

#### 二  上传代码到新分支

##### 1**$git status**:查看所在分支和修改的文件：

![](https://img-blog.csdn.net/20170104164803059?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

![]()

如上图，我所在的分支为wnaggenshen\_20170104,即为将要上传代码到该分支的地方；

（若所在分支不在要上传代码的分支，则使用 $git checkout  分支名，切换到该分支再进行操作）。

4个红色处表明已经被修改的文件，也为将要上传的文件。

#### 2 $ git add -A:添加所有要上传的文件(代码)

##### 

再$git status,此时4个文件变绿，表明将要上传到本地库；

##### 3 $ git commit -m  '这里随便写什么'

此时上传到本地repository；

![](https://img-blog.csdn.net/20170104164113206?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

![]()

##### 4 $ git push

此时提交到远程分支。

![](https://img-blog.csdn.net/20170104165023374?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvbm9hbWFuX3dncw==/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)

刷新Github，即可看到新上传的代码。
