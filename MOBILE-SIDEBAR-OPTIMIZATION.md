# 桑夏博客 - 移动端侧边栏优化

## ✅ 优化完成时间
2026-03-25 17:50

## 📱 优化内容

### 侧边栏隐藏式布局（抽屉式）

**桌面端（> 768px）**：
- 保持原有侧边栏布局
- 侧边栏固定在左侧

**移动端（≤ 768px）**：
- 隐藏固定侧边栏
- 显示汉堡菜单按钮（☰）
- 点击按钮弹出抽屉式侧边栏
- 侧边栏从左侧滑入（85% 宽度，最大 320px）
- 背景遮罩层，点击关闭
- 右上角关闭按钮（✕）

## 🔧 技术实现

### 1. HTML 结构
- 添加汉堡菜单按钮 `.menu-toggle`
- 添加抽屉容器 `.sidebar-drawer`
- 添加遮罩层 `.sidebar-drawer-overlay`

### 2. CSS 样式
- 抽屉固定定位，默认隐藏（left: -100%）
- 打开时滑入（left: 0）
- 遮罩层半透明背景
- 过渡动画 0.3s

### 3. JavaScript 功能
- `toggleSidebar()` - 开关侧边栏
- `updateDrawerStats()` - 更新抽屉统计数据
- 点击导航自动关闭侧边栏
- 点击遮罩层关闭侧边栏

## 🎨 可定制项

### 侧边栏宽度
```css
.sidebar-drawer {
    width: 85%;        /* 屏幕宽度的 85% */
    max-width: 320px;  /* 最大宽度 */
}
```

### 动画速度
```css
.sidebar-drawer {
    transition: left 0.3s ease; /* 0.3 秒平滑过渡 */
}
```

### 遮罩层透明度
```css
.sidebar-drawer-overlay {
    background: rgba(0,0,0,0.5); /* 50% 透明度 */
}
```
